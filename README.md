Remotion adaptation of the SpaceDust component at https://github.com/winkerVSbecks/3d-particle-effects-demo

## Commands

**Start Preview**

```console
npm start
```

**Render MP4 video**

```console
npm run build
```

**Upgrade Remotion**

```console
npm run upgrade
```
